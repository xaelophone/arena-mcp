What it does

A clean MCP interface for the Are.na API. Use it to browse, search, and update channels directly from clients that support MCP.

Read channels, blocks, and users as resources.

Search and traverse connections between blocks.

Create channels, create blocks, and manage connections.

Quick setup 
ARENA_ACCESS_TOKEN="YOUR_TOKEN"
npm exec --yes --package=github:xaelophone/arena-mcp arena-mcp
Client config example

This JSON is an example for Claude Desktop. Any MCP client can use this server, but the client-specific config format may differ. 

{
  "mcpServers": {
    "arena": {
      "command": "npx",
      "args": ["--yes", "--package=github:xaelophone/arena-mcp", "arena-mcp"],
      "env": { "ARENA_ACCESS_TOKEN": "YOUR_TOKEN" }
    }
  }
}
